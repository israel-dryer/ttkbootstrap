# Getting Started with ttkbootstrap 2

This hub organizes the core workflows from your first install all the way to the community channels that keep the project moving. Work through the guided steps or jump straight to the topic you need to keep shipping polished Tk interfaces.

![Getting started hero](https://placehold.co/800x800/FFFFFF/333333.webp?text=Getting%20Started%20Guide&font=lato)

## How to use this section

Follow the sequence or jump directly to the topic you need; the descriptions below explain what each page covers.

- **What is ttkbootstrap 2?** ? Understand the architecture, helpers, and patterns that shape the rest of the docs.
- **Why ttkbootstrap 2?** ? Learn about the major library upgrades, the new widget catalog, reactive signals, and richer localization support.
- **Quickstart (5-Minute App)** ? Install the package and launch a minimal themed window with interactive widgets.
- **Installation** ? Confirm Python 3.10+ support, run the recommended pip commands, and explore optional extras.
- **First Application** ? Assemble the App shell, layouts, and styled widgets that every ttkbootstrap 2 window shares.
- **Project Templates** ? Grab opinionated starters for minimal, themed, navigation, data, and localized experiences.
- **Learning Resources** ? Tutorial series, example apps, videos, and reference material to grow with the toolkit.
- **Community & Support** ? Where to ask questions, report issues, and contribute to the project.
