# Learning Resources

The learning resources page indexes the curated tutorials, examples, and videos that help you grow with ttkbootstrap 2.

![Resources](https://placehold.co/800x800/FFFFFF/333333.webp?text=Resources&font=lato)

## Tutorials

- **Building a form-heavy dashboard** — Focuses on inputs, validation, and responsive layout.
- **Feedback & dialogs deep-dive** — Examines Tooltip, Toast, and the dialog hierarchy (base, helpers, pickers).
- **Theming workshop** — Explains tokens, accent palettes, and how to switch themes at runtime.

## Example apps

- **Minimal UI kit** — A small demo showcasing input fields, data display widgets, and consistent `bootstyle` usage.
- **Task monitor** — Uses `Treeview`, `Meter`, and `FloodGauge` to visualize data with live updates.
- **Dialog playground** — Interactive page demonstrating each dialog type and helper from the Widgets catalog.

## Videos

- Embedded walkthroughs for quick installs, designing with `ttkbootstrap`, and shipping with PyInstaller.
- Recorded community hangouts that explore advanced layout patterns and CLI tooling.

## Stay current

- Follow the `#ttkbootstrap` tag on GitHub Discussions for new tutorials.
- Check the `Showcase` section for community-contributed apps and templates.
