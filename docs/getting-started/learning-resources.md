# Learning Resources

Use these curated tutorials, examples, and videos to deepen your understanding of ttkbootstrap 2 and the design system it exposes.

![Resources](https://placehold.co/800x800/FFFFFF/333333.webp?text=Resources&font=lato)

## Tutorials

- **Building a form-heavy dashboard** walks through inputs, validation, PageStack navigation, and FormDialog automation.
- **Feedback & dialogs deep dive** covers Tooltips, Toasts, Dialog helpers, and how to layer message boxes over responsive layouts.
- **Theming workshop** explains Bootstyle tokens, theme switching at runtime, and how to extend palettes with custom ThemeProvider instances.

## Example apps

- **Minimal UI kit** shows how to combine TextEntry, PasswordEntry, and SelectBox controls with consistent Bootstyle styling.
- **Task monitor** demonstrates Meter, FloodGauge, TableView, and signal-driven progress updates.
- **Dialog playground** is an interactive page that samples each dialog type, the Form widget, and the new ContextMenu helpers.

## Videos & demos

- Short walkthroughs that cover installation, theming, layout, and shipping with PyInstaller.
- Community hangouts that explore advanced `ttkbootstrap 2` patterns, theme authoring, and signal-based data binding.
- Live-coding sessions that follow the Getting Started flow from Install through a first app and beyond.

## Reference materials

- The Widgets catalog organizes controls by category and lists Bootstyle-ready examples for each widget.
- The Design system section documents colors, typography tokens, variants, and localization patterns in one place.
- The Guides highlight runtime configuration, theming tips, signals, events, and packaging workflows for production apps.

## Stay current

- Watch the Showcase for new templates and community projects.
- Follow the repo discussions for announcements, RFCs, and release notes.
- Contribute guides, translate docs, or submit templates to keep the ecosystem growing.
