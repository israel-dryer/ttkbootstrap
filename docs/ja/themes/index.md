# テーマ

ttkbootstrapには、美しくデザインされた多数のライトテーマと
ダークテーマが同梱されています。これらは、ttkbootstrapをインストールした後に
ターミナルで以下のコマンドを入力することで、デモとして確認できます。

```shell
>>> python -m ttkbootstrap
```

以下の画像のようなデモ画面が表示されます。**Select a theme** コンボボックスで
テーマを選択すると、デモ画面が新しいテーマに更新されます。

また、[ライトテーマ](light.md)や
[ダークテーマ](dark.md)のスクリーンショットも確認できます。
