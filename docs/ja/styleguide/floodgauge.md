# 水位計

オプションで表示テキストを設定できるプログレスバーです。

このウィジェットスタイルは、**プライマリ**カラー（デフォルト）
または[選択した色](index.md#colors)のインジケーターを備えています。谷間の色は、
インジケーターの色を彩度を下げたバリエーションです。 

![floodguage](../assets/widget-styles/floodgauge.gif)

```python
# デフォルトのFloodgaugeスタイル
Floodgauge()

# 成功色を使用したFloodgaugeスタイル
Floodgauge(bootstyle="success")
```
