[Bootstrap](https://getbootstrap.com/)にインスパイアされた、モダンなフラットスタイルのテーマをオンデマンドで適用できる、tkinter用の高性能テーマ拡張機能です。

## 📦 機能

✔️ [**組み込みテーマ**](themes/index.md)   
厳選された10種類以上の[ダーク](themes/dark.md)および[ライト](themes/light.md)テーマ  

✔️ [**定義済みスタイル**](styleguide/index.md)  
**アウトライン**ボタンや**丸型トグル**ボタンなど、
美しい[定義済みウィジェットスタイル](styleguide/index.md)が多数用意されています。

✔️ [**シンプルなキーワードAPI:**](gettingstarted/tutorial.md#use-themed-widgets)  
**primary**や**striped**といった[シンプルなキーワード](gettingstarted/tutorial.md#use-themed-widgets)を使って色やタイプを適用できます。
従来の**primary.Striped.Horizontal.TProgressbar**といった従来の方法の代わりに、[シンプルなキーワード](gettingstarted/tutorial.md#use-themed-widgets)
（**primary**や**striped**など）を使用して色やスタイルを適用します。Web開発でBootstrapを使用したことがある方なら、
CSSクラスを使ったこのアプローチにはすでにお馴染みでしょう。

✔️ [**多数の新しいウィジェット:**](api/widgets/dateentry.md)  
ttkbootstrap には、[Meter](api/widgets/meter)、[DateEntry](api/widgets/dateentry.md)、
[Floodgauge](api/widgets/floodgauge) など、美しくデザインされた新しいウィジェットがいくつか含まれています。 さらに、[ダイアログ](api/dialogs/dialog.md) 
もテーマ化され、完全にカスタマイズ可能になりました。

✔️ [**組み込みのテーマ作成ツール:**](themes/themecreator.md)  
独自のテーマを作成したいですか？簡単です！ttkbootstrapには、
独自のカスタムテーマを簡単に作成、
読み込み、探索、適用できる組み込みの
[テーマ作成ツール](themes/themecreator.md)が含まれています。
