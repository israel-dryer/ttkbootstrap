Treeview
========

A multi-column tree and table view.

bootstyle → ttk style name
--------------------------

Any of the accent colors -- ``primary``, ``secondary``, ``success``, ``info``, ``warning``, ``danger``, ``light``, ``dark`` -- substitutes for ``primary`` above. ``neutral`` does not apply to this family.

Hand-styling example
--------------------

.. code-block:: python

   import ttkbootstrap as ttk

   app = ttk.App()
   style = app.style

   style.configure("my.Treeview", background="#3498db", foreground="white")
   ttk.Treeview(app, style="my.Treeview").pack(padx=8, pady=8)

   app.mainloop()
