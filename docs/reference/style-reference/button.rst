Button
======

A clickable action trigger.

bootstyle → ttk style name
--------------------------

Any of the accent colors -- ``primary``, ``secondary``, ``success``, ``info``, ``warning``, ``danger``, ``light``, ``dark`` -- substitutes for ``primary`` above. ``neutral`` also applies here.

.. list-table::
   :header-rows: 1
   :widths: 50 50

   * - ``bootstyle``
     - ttk style name
   * - ``primary``
     - ``primary.TButton``
   * - ``primary ghost``
     - ``primary.Ghost.TButton``
   * - ``primary link``
     - ``primary.Link.TButton``
   * - ``primary outline``
     - ``primary.Outline.TButton``

Hand-styling example
--------------------

.. code-block:: python

   import ttkbootstrap as ttk

   app = ttk.App()
   style = app.style

   style.configure("my.TButton", background="#3498db", foreground="white")
   ttk.Button(app, text="Custom", style="my.TButton").pack(padx=8, pady=8)

   app.mainloop()
