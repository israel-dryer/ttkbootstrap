Toggle
======

A switch-style checkbutton (round or square).

bootstyle → ttk style name
--------------------------

Any of the accent colors -- ``primary``, ``secondary``, ``success``, ``info``, ``warning``, ``danger``, ``light``, ``dark`` -- substitutes for ``primary`` above. ``neutral`` does not apply to this family.

.. list-table::
   :header-rows: 1
   :widths: 50 50

   * - ``bootstyle``
     - ttk style name
   * - ``primary-toggle``
     - ``primary.Toggle``
   * - ``primary-round-toggle``
     - ``primary.Round.Toggle``
   * - ``primary-square-toggle``

Hand-styling example
--------------------

.. code-block:: python

   import ttkbootstrap as ttk

   app = ttk.App()
   style = app.style

   style.configure("my.Toggle", background="#3498db", foreground="white")
   ttk.Checkbutton(app, text="Custom", style="my.Toggle").pack(padx=8, pady=8)

   app.mainloop()
