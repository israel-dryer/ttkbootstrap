Frame
=====

A plain container surface.

bootstyle → ttk style name
--------------------------

Any of the accent colors -- ``primary``, ``secondary``, ``success``, ``info``, ``warning``, ``danger``, ``light``, ``dark`` -- substitutes for ``primary`` above. ``neutral`` does not apply to this family.

.. list-table::
   :header-rows: 1
   :widths: 50 50

   * - ``bootstyle``
     - ttk style name
   * - ``primary``
     - ``primary.TFrame``

Configurable options
--------------------

Options you can set with ``style.configure(...)`` / ``style.map(...)``:

``bordercolor``, ``borderwidth``, ``darkcolor``, ``lightcolor``, ``relief``

Supported states
----------------

This style declares no ``style.map(...)`` overrides; any state-dependent appearance is drawn from its image elements.



Hand-styling example
--------------------

.. code-block:: python

   import ttkbootstrap as ttk

   app = ttk.App()
   style = app.style

   style.configure("my.TFrame", background="#3498db", foreground="white")
   ttk.Frame(app, style="my.TFrame").pack(padx=8, pady=8)

   app.mainloop()
