Hand-styling example
--------------------

.. code-block:: python

   import ttkbootstrap as ttk

   app = ttk.App()
   style = app.style

   style.configure("my.TNotebook", background="#3498db", foreground="white")
   ttk.Notebook(app, style="my.TNotebook").pack(padx=8, pady=8)

   app.mainloop()
