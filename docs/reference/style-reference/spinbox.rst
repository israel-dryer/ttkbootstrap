Spinbox
=======

Hand-styling example
--------------------

.. code-block:: python

   import ttkbootstrap as ttk

   app = ttk.App()
   style = app.style

   style.configure("my.TSpinbox", background="#3498db", foreground="white")
   ttk.Spinbox(app, style="my.TSpinbox").pack(padx=8, pady=8)

   app.mainloop()
