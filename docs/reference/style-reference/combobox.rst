Hand-styling example
--------------------

.. code-block:: python

   import ttkbootstrap as ttk

   app = ttk.App()
   style = app.style

   style.configure("my.TCombobox", background="#3498db", foreground="white")
   ttk.Combobox(app, style="my.TCombobox").pack(padx=8, pady=8)

   app.mainloop()
