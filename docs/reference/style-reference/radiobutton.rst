Hand-styling example
--------------------

.. code-block:: python

   import ttkbootstrap as ttk

   app = ttk.App()
   style = app.style

   style.configure("my.TRadiobutton", background="#3498db", foreground="white")
   ttk.Radiobutton(app, text="Custom", style="my.TRadiobutton").pack(padx=8, pady=8)

   app.mainloop()
