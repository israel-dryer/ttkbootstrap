Hand-styling example
--------------------

.. code-block:: python

   import ttkbootstrap as ttk

   app = ttk.App()
   style = app.style

   style.configure("my.TCheckbutton", background="#3498db", foreground="white")
   ttk.Checkbutton(app, text="Custom", style="my.TCheckbutton").pack(padx=8, pady=8)

   app.mainloop()
