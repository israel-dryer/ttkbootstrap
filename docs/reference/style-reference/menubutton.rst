Menubutton

Hand-styling example
--------------------

.. code-block:: python

   import ttkbootstrap as ttk

   app = ttk.App()
   style = app.style

   style.configure("my.TMenubutton", background="#3498db", foreground="white")
   ttk.Menubutton(app, text="Custom", style="my.TMenubutton").pack(padx=8, pady=8)

   app.mainloop()
