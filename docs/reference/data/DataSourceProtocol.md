# DataSourceProtocol

Protocol defining the interface for datasource implementations.

All datasources implement this protocol: [MemoryDataSource](MemoryDataSource.md), [SqliteDataSource](SqliteDataSource.md), [FileDataSource](FileDataSource.md). For a base implementation, see [BaseDataSource](BaseDataSource.md).
