API Reference
=============

The complete reference for the public API ttkbootstrap authors, generated from
the source docstrings. Native ttk widgets have **no** API page here — for their
plain constructor options, see the standard :mod:`tkinter.ttk` reference; their
styling surface is in the :doc:`Style Reference </reference/style-reference/index>`.

.. note::

   This section is being built out. The spike below documents the shipped
   widgets to prove the autodoc pattern; the engine, toolkit, dialogs, and
   utility pages follow in later slices.
